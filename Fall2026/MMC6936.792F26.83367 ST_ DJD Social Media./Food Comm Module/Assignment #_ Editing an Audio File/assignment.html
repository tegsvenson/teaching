Assignment #_ Editing an Audio File

Points: 20.0 | Due: None

Module #: Assignment # | Editing an Audio File

  Assignment Instructions 

1. Download Adobe Audition and Login

Through the Adobe Creative Cloud app on your computer, open and/or download Adobe Audition. 

If you do not already have Adobe Creative Cloud access, request it through IT.

2. Download the Assignment File

 For this assignment, you are to use the file provided. Follow the instructions in the template.

Press the three-dot menu to access the download button.

 

3. Utilize the Four Lessons to Complete the Assignment

You may use your own judgment in utilizing the tool taught in this module to edit your audio file.

  • The noise should be noticeably reduced.
  • Any pops should be removed.
  • Any hisses should be removed.
  • The audio should be normalized to -3db.

4. Export Your File

File > Export > File... > Save to PC folder

Name your file something easily identifiable, so you don't submit the unedited version (i.e. Lastname_assignment#_audio_editted).

You should export to both a WAV file and MP3 file.

4. Turn it In

When you have completed the assignment, submit the WAV & MP3 files here.

Deadline: You have until 11:59 pm on March 24 to submit your file.
