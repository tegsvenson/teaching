  The Objectives

Here is a brief overview of the Media Diary assignment. This assignment is the major assignment for the first half of this course. This assignment is a part of this course to help you:

  • Increase awareness of the influence of media technology in our daily lives.
  • Practice inferring insights from quantitative information.
  • Apply theories related to user experience in reality.

To accomplish these objectives, you will collect data about your media use for a week, reflect on them on a daily basis, and come up with a comprehensive analysis of your own user behavior.

  The Procedure

The following is the simplified steps to complete this assignment series. You can find more detailed information in each page.

  • Prepare Your Device & Start Data Collection (August 27): This is a 0-point assignment that prompts you to set your device up for Media Diary.
  • Post Your Daily Journal (TBA): Post the required screen shots and your reflections for a week. Ideally, you should make an entry every day.
  • Concluding Report (TBA): Write up a formal document culminating your findings.
