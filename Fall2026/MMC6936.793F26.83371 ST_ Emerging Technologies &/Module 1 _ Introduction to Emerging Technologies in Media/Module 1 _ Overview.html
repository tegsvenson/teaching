Module 1 Overview

The purpose of this module is to introduce you to the different phases within the course and how the ideas in the lectures and readings are related to each module’s activities and the overarching project. 

 Module Learning Outcome

Upon successful completion of this module, you will be able to:

  1. Critique communication or media interfaces, applications, and/or services from a user perspective.

  2. Identify problems and solutions regarding technology in communication.

    • Assignments and learning activities used to assess Module Learning Outcome: (1)  Introduction Discussion;

  3. Discuss communication perspectives regarding user-to-user and application.

    • Assignments and learning activities used to assess Module Learning Outcome: (1) Introduction Discussion;
