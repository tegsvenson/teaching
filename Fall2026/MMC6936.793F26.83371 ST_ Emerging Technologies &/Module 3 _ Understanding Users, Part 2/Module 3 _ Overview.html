Module 3 | Overview

This module is the second half of the two part module that provides the fundamental concepts and perspectives for understanding emerging interactive media. We will focus more on the practical and central ideas and considerations related to designing interfaces for communication devices.

 Module Learning Outcome

Upon successful completion of this module, you will be able to:

  1. Describe the core psychological underpinnings and its concepts related to user experience in emerging media technologies.
    • Assignments and learning activities used to assess Module Learning Outcome: (1) Quiz, (2) Assignment 
  2. Analyze effective user communication from the perspective of other users.
    • Assignments and learning activities used to assess Module Learning Outcome: (1) Quiz, (2) Assignment 
  3. Apply theoretical and practical principles to strategically design a mediated experience for existing or novel applications.
    • Assignments and learning activities used to assess Module Learning Outcome: (1) Quiz, (2) Assignment
