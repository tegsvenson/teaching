Module 11 Overview

In the final module of the course, you will test the prototype, compile the project assignment, and create the final report.

 Module Learning Outcome

Upon successful completion of this module, you will be able to:

  1. Identify problems and solutions concerning technology in communication. 
    • Assignments and activities used to assess Module Learning Outcome: (1) final project;
  2. Apply theoretical and practical principles to strategically design a mediated experience for existing or novel applications. 
    • Assignments and activities used to assess Module Learning Outcome: (1) final project;
  3. Utilize prototyping tools to apply and test mediated experience for existing or novel applications. 
    • Assignments and activities used to assess Module Learning Outcome: (1) final project;
  4. Apply methods and tools for conducting a qualitative approach to gathering and analyzing data about user experiences. 
    • Assignments and activities used to assess Module Learning Outcome: (1) final project;
  5. Apply methods and tools for conducting a quantitative approach to gathering and analyzing data about user experiences. 
    • Assignments and activities used to assess Module Learning Outcome: (1) final project;
