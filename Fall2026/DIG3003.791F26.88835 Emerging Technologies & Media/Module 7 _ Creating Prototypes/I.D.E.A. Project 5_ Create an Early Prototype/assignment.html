IDEA 5 | Create an Early Prototype

 Assignment Instructions

Create an early prototype. We learned about three different steps in creating prototypes - wireframes, mockups, and prototypes. They vary in their visual fidelity and interactivity. Here are the requirements:

  1. Expect to have at least the wireframe finished.
  2. All the functions, and each page and step should have a visual. It can still be a pen-and-paper drawing, but all required elements should have their own image.
  3. Clearly explain how each visual is connected to each other. In other words, explain the interactivity and how the follow the task you described in your project description assignment.
  4. Put together the images into a PDF and submit.
  5. If you have an interactive version ready, that means you are moving ahead. Please share that version as a comment under your submission.
